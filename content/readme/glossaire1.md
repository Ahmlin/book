```{glossary}
Table de vérité
  Une table de vérité est une façon de représenter à l'écrit l'état d'entrée et de sortie d'un circuit électrique.  

Représentation de l'information
  En informatique, on appelle représentation de l'information l'étude de notions liées au codage, au cryptage et au stockage des données.  

Ordinateur
  Un ordinateur est une machine composée de circuits électriques qui est capable d'effectuer des calculs et de stocker de l'information sous forme binaire. 

Numérisation
  La numérisation est le procédé par lequel une information extraite du monde réel est transformée en nombres. 

Stockage
  En informatique, on appelle stockage les techniques utilisées pour retenir une information à l'intérieur d'un circuit électrique, que celui-ci soit alimenté ou non. 

Transmission
  En informatique, on appelle transmission l'interaction entre un émetteur et un récepteur qui s'échangent une information. Exemple : le bluetooth est un mode de tranmission de l'information.

Traitement 
  Le traitement des données englobe les techniques et les pratiques consistant à transformer, manipuler, stocker, ou envoyer des donées informatiques. 

Systèmes de communication
  Un système de communication est un ensemble d'infrastucture, de codes, d'alphabets, de pratiques, qui permettent à un émetteur d'envoyer un message à un récepteur. Exemple : Internet. 

Signal 
  Un signal est un objet utilisé pour en représenter un autre. 

Télégraphe
  Télégraphe est le terme générique pour définir un système de communication et un mode de tranmission de l'information d'un point A à un point B. Du grec "télé" (à distance) et "graphein" (écrire)

Codex 
  Un codex est un dictionnaire permettant de traduire des signaux d'un langage à un autre. 

Standardisation
  La standardisation est la pratique consistant à établir une norme partagée par une communauté, servant de vocabulaire commun. 

Code binaire
  Le code binaire est la fonction bijective allant des nombres décimaux aux nombres binaires. Lol.

Transistor
  Un transistor est un composant électronique permettant de contrôler l'ouverture ou la fermeture du passage des électrons d'un point A à un point B. 

Code
  En informatique, un code est un alphabet traduisant des signaux d'un language à un autre. 
```
